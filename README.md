# 🌟 AI Plugin Suite (大模型效率增强插件系列)

一个致力于重塑网页生产力的下一代 AI 插件集成系列，支持 Microsoft Edge、Google Chrome 等主流浏览器。
我们为市面上主流的大语言模型（LLM）网页端提供统一、专业、高效的界面增强与导航体验。

## 🚀 核心愿景与通用功能

随着 AI 对话越来越长，信息的定位和回顾变得异常困难。本系列插件旨在解决超长对话带来的“迷航”困扰，均具备以下通用特性：

* **📍 动态大纲生成**：实时扫描对话内容，自动提取关键指令作为侧边栏目录。
* **🔄 双向滚动同步**：页面滚动时侧边栏自动跟踪高亮；点击大纲，页面平滑跳转。
* **🔍 智能内容过滤**：内置搜索框，支持在数十条对话中通过关键词一键定位。
* **🛡️ 纯净且安全**：采用 Shadow DOM 技术完美融入原生 UI，不污染原页面；纯本地运行，**绝不收集或上传**任何对话数据。

## 🛠️ 安装说明 (如何使用)

目前本系列插件代码完全开源，你可以通过浏览器的“开发者模式”在本地极速安装体验。

### 第一步：获取代码
点击本仓库右上角的绿色的 **"Code"** 按钮，选择 **"Download ZIP"**，下载压缩包后解压到你的电脑上。
*(或者使用 Git 命令：`git clone 你的仓库地址`)*

### 第二步：在浏览器中加载插件

**如果您使用的是 Microsoft Edge：**
1. 在浏览器地址栏输入 `edge://extensions/` 并回车。
2. 打开页面左侧（或左下角）的 **“开发人员模式 (Developer mode)”** 开关。
3. 点击页面上方的 **“加载解压缩的扩展 (Load unpacked)”** 按钮。
4. ⚠️ **关键步骤**：在弹出的文件选择窗口中，不要直接选择总文件夹，而是**选择你想要安装的具体插件所在的文件夹**（例如：选择 `gemini` 这个文件夹）。

**如果您使用的是 Google Chrome：**
1. 在浏览器地址栏输入 `chrome://extensions/` 并回车。
2. 打开页面右上角的 **“开发者模式 (Developer mode)”** 开关。
3. 点击左上角的 **“加载已解压的扩展程序 (Load unpacked)”** 按钮。
4. ⚠️ **关键步骤**：同样，选择对应的子文件夹（如 `gemini` 文件夹）。

### 第三步：开始体验
安装成功后，打开对应的 AI 网页（例如访问 Gemini），稍等片刻，插件将自动运行。你会看到页面右侧出现专属的悬浮球或导航栏，尽情体验高效对话吧！
